# Code of Conduct

## Our Standards

We want PrizmKit to be a respectful, useful, and technically rigorous community. Participants should:

- communicate with patience and respect;
- focus criticism on ideas, code, and evidence rather than people;
- explain trade-offs and uncertainty clearly;
- welcome different levels of experience and different AI host platforms;
- protect private information, credentials, and user project data.

Unacceptable behavior includes harassment, discrimination, threats, deliberate exposure of private information, malicious use of project artifacts, and disruptive bad-faith participation.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation when behavior violates these standards. Report conduct concerns privately to the maintainers with relevant context and links.

## Scope

This code applies in project spaces and whenever someone represents the PrizmKit project or community.
